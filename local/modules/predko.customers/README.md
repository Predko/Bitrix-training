  Модуль Customers предназначен для ведения учёта: 
  - оказанных услуг индивидуальным предпринимателем (class ContractsTable),
  - поступлений на счёт ИП (class IncomeTable),
  - расходов (class ExpencesTable),
  - списка клиентов ИП (class CustomersTable).

  В состав будут входить компоненты для:
  - импорта данных из банковской выписки в формате CSV,
  - экспорта данных в формате CSV,
  - отдельные компоненты для отображения и ввода информации таблиц Customers, Contracts, Income, Expences 
    с необходимой фильтрацией и обработкой информации,
  - компонент создание договора/акта выполненных работ с выводом на печать,
  - комплексный компонент, для отображения списка клиентов, с выводом договоров и оплаты для выбранного клиента.
